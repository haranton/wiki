Базовые типы данных

Числовые: `int`, `int8`, `int16`, `int32`, `int64`

#### Целые беззнаковые

| Тип       | Диапазон                                          |
|-----------|---------------------------------------------------|
| `uint8`   | 0 … 255 (`byte` -- псевдоним)                     |
| `uint16`  | 0 … 65,535                                        |
| `uint32`  | 0 … 4,294,967,295                                 |
| `uint64`  | 0 … 18,446,744,073,709,551,615                    |
| `uint`    | зависит от архитектуры                            |
| `uintptr` | используется для хранения указателей в виде целых |

Числа с плавающей точкой: `float32`, float64

Комплексные числа: complex64

#### Строки и символы

-  `string` -- неизменяемая последовательность байт (UTF-8)

-  `rune` -- отдельный символ Unicode (по сути `int32`)

-  byte - псевдоним для `uint8`, часто используется для байтов
